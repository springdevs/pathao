== Installation ==

= Installation from within WordPress =

        1. Visit 'Plugins > 'Add New'.
        2. Search for 'Pathao for Woocommerce'.
        3. Install and activate the 'Pathao for Woocommerce' plugin.

    = Manual installation =

        1. Upload the entire `Pathao for Woocommerce` folder to the `/wp-content/plugins/` directory.
        2. Visit 'Plugins'.
        3. Activate the `Pathao for Woocommerce` plugin.
